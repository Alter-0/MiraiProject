# MiraiProject

这是一个番剧在线播放网站  
只作为团队学习项目，并不投入使用
----
##MiraiProject目录树

├─animate  番剧页  
├─api  看板娘使用api  
├─auth  用户相关界面  
├─css  储存通用css样式  
├─home 导航页  
├─image 储存图片  
├─index 索引页  
│  ├─conn  
│  └─images  
├─js 通用js脚本  
├─live2d 看板娘相关配置  
├─main 主页  
│  └─image  
└─res 资源文件
